（文献阅读）通用的 paraphrastic sentence embedding

发表于 2018-05-16

句子建模相关文献阅读

文献名称：Towards Universal Paraphrastic Sentence Embeddings

摘要（ABSTRACT）

本文研究了从释义数据库中有监督地学习通用释义句嵌入（paraphrastic sentence embeddings）的问题。本文比较了六种结构，根据标注文本相似性数据集对它们进行评价，这些数据集来自与训练数据相同的分布和广泛的其他领域。结果表明复杂的结构，比如 LSTM 在 in-domain 数据上表现最好；但是，在 out-of-domain 场景下，简单的结构，比如 word averaging 反而优于 LSTMs。本文设计的 averaging model 甚至与针对特定任务调优过的模型性能相当，但更简单易用。

为了更好地理解这些结构是如何比较的，本文在三个有监督的 NLP 任务上进行了实验：句子相似度（sentence similarity）、蕴涵（entailment）和情感分类（sentiment classification）。结果表明，词平均模型在句子相似性和 entailment 任务上表现优于 LSTMs。但是在情感分类任务上，LSTMs 表现非常好，甚至在斯坦福 Treebank 数据集上达到了 state-of-art。

此外，本文演示了如何将预训练的 sentence embedding 与这些有监督任务结合起来，并将它们用作先验（prior）和黑盒（black box）特征提取器，这使得在 SICK similarity 和 entailment 任务上达到 state-of-art。本文公开了所有资源在 http://ttic.uchicago.edu/˜wieting （无法打开，但是可以在作者的 GitHub 上找到源码）

文献名称: Better Text Understanding Through Image-To-Text Transfer

摘要（Abstract）

泛型文本嵌入（Embedding）已经成功地用于各种任务中。然而，它们往往是通过从纯文本语料库中捕获共现结构来学习的，从而限制了它们的泛化能力。本文探索了将视觉信息纳入文本表示的模型。基于 comprehensive ablation studies（控制变量法），我们提出了一个概念简单，但性能良好的结构。在一套既定的基准上，它优于以往的多模态方法。我们还改进了图像相关文本数据集（image-related text datasets）的最新成果，使用数量级较少的数据。

（经典算法问题）鹰蛋问题

发表于 2018-05-13

经典的动态规划问题

问题描述

教授手上有k个一模一样的鹰蛋，教授想研究这些蛋的硬度F，测试方法是将蛋从高为n层的楼上不断自由落下；每个蛋在F+1层及以上掉下都会碎，而在F层及以下不会碎；每个蛋可以重复测试直到它碎了为止。

例如：蛋从第1层掉下碎了，则F=0；蛋从第N层掉下未碎，则F=N。

求在给定k和n下为了确定F在最坏情况下需要测试的最少次数。

示例：k=1, n=10，则ans=10
