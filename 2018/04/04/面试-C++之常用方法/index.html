Python 中是这样的

C++ 虽然也提供了 replace 方法，但逻辑不一样：它需要你提供被替换字符串的首位置和待替换子串的长度，然后替换，且只替换一次

特别的，如果是对 char 替换，那么考察重点往往不是替换本身，而是 char 作为数组，内部元素的移动（要求不使用额外空间）；而且替换的套路也类似，先统计待替换子串的数量，算出替换后的总长度，利用双指针从后往前替换，还有别忘了手动加上 ‘\0’
