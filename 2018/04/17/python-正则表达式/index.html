python 正则表达式记录

发表于 2018-04-17

“如果你想用正则表达式来解决问题，那么你面临的问题将会变成两个！”

正则表达式是个好东西，但是每次想用它的时候都感到头大。

这里简单记录一下 Python 中 re 模块的基本用法，以作备忘。

前言

学习正则的关键是，以完成任务为目标；想要完全掌握正则，可以去看看那本《正则表达式》的厚度清醒一下。

正则的强大之处不多赘述，最基本的用法是“模式匹配/查找”，这在爬虫里面用的比较多。

而我主要用正则来清洗文本数据，所以，对我来说正则最常用的功能是两个：

  1. 保留你需要的
  2. 去掉你不要的

这两个可以看做是一个任务，但是分开来处理会方便很多。

re 基本语法

下表列出了常用的 re 语法。如果你提供了 flag 参数，某些元素的含义可能会改变。

模式 描述
. 通配符；匹配除换行符外的任意字符，如果 flag=re.DOTALL 则包括换行符
^ 匹配字符串的开头
$ 匹配字符串的末尾
* 匹配 0 次或多次
+ 匹配 1 次或多次
{n} 匹配 n 次
{m, n} 匹配 m 到 n 次，贪婪匹配
? 非贪婪匹配
[...] 匹配在[]中的字符；如 [amk] 匹配 ‘a’，’m’或’k’；可以用 - 表示一个范围，如 [a-zA-Z] 匹配所有英文字母、[\u4e00-\u9fa5] 匹配所有中文
[^...] 匹配不在 [] 中的字符
() 匹配括号内的表达式，表示一个组
\w 匹配字母数字及下划线
\W 匹配非字母数字及下划线
\s 匹配任意空白字符，等价于 [\t\n\r\f]
\S 匹配任意非空字符
\d 匹配任意数字，等价于 [0-9]
\D 匹配任意非数字

数据清洗时最常用的就是 [...] 和 [^...]，使用示例

注意：正则是按单个字符依次匹配的，所以类似 \w 的转义也是只针对一个字符

完整的正则表达式语法，参考 Python3 官方文档；也可以参考 Python2 文档，比 Python3 的版本排版更清晰

re 模块的常用函数

用途 函数声明 返回值
编译 re.compile(pattern, flags=0) re.RegexObject
查找 re.search(pattern, string, flags=0) re.MatchObject
查找 re.findall(pattern, string, flags=0) list
查找 re.finditer(pattern, string, flags=0) iterator
替换 re.sub(pattern, repl, string, count=0, flags=0) string
分割 re.split(pattern, string, maxsplit=0, flags=0) list

其中，比较常用的是 re.sub、re.split 和 re.findall。

re.compile 返回的对象 re.RegexObject 也支持 re 模块下的大部分函数；也就是说，你可以这样调用

使用 re.compile 会提前编译给定的 RE，避免每次都重新编译；不过 Python 也没有那么蠢，它会缓存编译过的 RE。

re.match()的用途比较窄，一般只用来做完全匹配，关于它与 re.search，参考 search() vs. match()

因为几乎用不到它，所以最好别记住它，不然你每次都会纠结它跟 re.search 有什么区别。
