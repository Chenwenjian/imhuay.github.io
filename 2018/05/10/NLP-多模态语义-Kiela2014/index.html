摘要

本文提出了一种多模态概念表示（multi-modal concept representations），通过连接 skip-gram
linguistic representation vector 和 visual concept representation vector。这种转移学习方法比传统的 bag-of-visual-word 方法带来了明显的性能提升。实验结果在 WordSim353 和 MEN 语义关系评价任务中得到了验证。本文使用的 visual features 来自 ImageNet 或 ESP Game images。

前言

以前的多模态方法一般包含两各特征：1. 语言学特征（linguistic feature），2. 来自传统 CV 技术的视觉特征（visual features），比如 SIFT 特征，视觉词袋模型（bags of visual words）

Bag of visual words(Bag of Words(BOW)模型) - CSDN博客

随着迁移学习和 CNN 在 CV 领域的发展，很多任务开始使用卷积层来抽取图像特征，作为任务的 mid-level feature extractors。更快而且效果更好。

本文使用了 CNN-extracted 的视觉特征来构建多模态语义表示模型。并得到了一些有趣的结果：

  1. 加入视觉特征（组成多模态特征）能有效提升单独使用语义特征的性能；
  2. 性能提升的原因是因为加入了”the information contained in the images”，而不是”the information used to select which images to use to represent a concept”；
  3. CNN-extracted 的特征具有一个有趣的性质；
  4. 本文认为这项工作代表了第一个多模态分布语义学的方法，它完全依赖于对语言和视觉的深入学习。

相关工作

多模态分布式语义（Multi-Modal Distributional Semantics）

多模态模型的动机是人类在学习概念时是并行的。人类的语义知识很大程度上依赖于感知信息(Louwerse, 2011)。事实上，大量研究表明，许多概念是建立在知觉系统（perceptual system）之上的(Barsalou, 2008)。

认知（cognition）

在分布式语义（distributed semantics）的背景下，实现多模态的方法之一就是将来自语义的特征/特征与其他模式的特征进行组合，比如来自图像的特征。一些研究已经取得成果 (Bruni et al., 2014)

多模态深度学习（Multi-modal Deep Learning）

介绍不同的多模态表示学习方法：

  • 受限玻尔兹曼机 (Srivastava and Salakhutdinov, 2012; Feng et al., 2013)
  • 自编码器 (Wu et al., 2013; Silberer and Lapata, 2014)
  • RNN (Socher et al., 2014)

多模态模型的应用

  • crossmodal tasks (Lazaridou et al., 2014)

本文的模型保持了模式的分离，并遵循标准的多模态方法，在一个单一的语义空间模型中将语言和视觉表示连接在一起。这么做的优点是可以为不同的模式提供单独的数据来源。

本文直接从图像中学习表示，而不是以更高层次的表示作为起点（比如视觉词袋模型，BOVW）

深度卷积网络

最近的一系列研究表明，从 CNN 网络中提取的图像特征（image embedding, visual embedding）非常有效，在很多任务上超过了之前最好的模型 (Razavian et al., 2014)

视觉特征（Perceptual Representations）

标准/传统的多模态表示中的视觉部分一般都是使用的 BOVW 特征（bag-of-visual-words，视觉词袋模型） (Sivic and Zisserman, 2003)

本文使用 BOVW 作为 Baseline。

本文使用的预训练 CNN 网络是 AlexNet（8层） (Krizhevsky et al., 2012)，使用前 7 层（CNN）来抽取图像特征

VGG(2014), GoogLeNet(2014), ResNet(2015), …

对于某个CV的问题，选一个优秀的核心网络作为基础，然后 fine-tune 是一个常见的思路

本文考虑了两种方法来处理（aggregate）每张图的特征向量

  1. CNN-Mean: computes the average of all feature vectors
  2. CNN-Max: computes the component-wise maximum of all feature vectors.

实验结果表明法2更好，因为来自该网络的特征向量相当稀疏（sparse），只有 22% 左右的非 0 系数；此外使用 Max 方法可以解释为 bags of visual properties。

语义特征（Linguistic representations）

100 维的词向量（skip-gram）

多模态特征（Multi-modal Representations）

先对两个特征向量做 L2 正则化，然后做加权拼接

$$ v_{concept} = [\alpha v_{visual}; (1-\alpha) v_{ling}]
$$

实验设置

本文分别在 ImageInt 和 ESP Game 数据集上生成了 image embedding。

然后在两个公开的语义关联数据集上测试了得到的多模态语义特征。

图像数据

ImageInt

ImageInt (Deng et al., 2009) 是一个按照 WordNet 层次结构组织的大规模图像数据集；

本文使用了 22000 个 synsets 共 12.5M 张图片

WordNet 2014 有大概 117000 个 synsets

ESP Game

ESP Game 数据集 (Von Ahn and Dabbish, 2004) 包含 100000 张图片，但是每张图片有平均 14 个标签（esp 中的图像不区分前景中的对象和背景中的对象，也不区分对象的相对大小），共 20515 个单词

评价方法

本文使用用两个语义关联数据集（WordSim353 和 MEN）来评估多模态词表示。

衡量指标：spearman相关系数 + 余弦相似度

实验分析

  • Multi-modal 的效果
  • Image embedding 的贡献——性能的提升是由于图像本身还是由于文字标签的关联信息
  • 不同的图像数据集对性能的影响

  • CNN-Mean 和 CNN-Max 在 W353 和 MEN 上有不同的表现

    • WordSim353 captures both similarity(tiger-cat) and relatedness(Maradona-football)

    • MEN capture semantic relatedness only

    • CNN-Max using sparse feature vectors means that we treat the dominant components as definitive of the concept class, which is more suited to similarity.

    • CNN-Mean averages over all the feature components, and as such might be more suited to relatedness.

  • 权重参数 $\alpha$ 对于结果的影响

  • 抽象词无法从 image embedding 中获益

  • CNNs 明显比 BoVW 有效

错误分析

这一节讲的是系统在测试中比较失败的案例，通过对比效果最好的案例来寻找打分错误的原因

本文选了最好和最差的 5 个来对比

从表格来看，最好与最差的定义是通过与 gold standard 的差值来定义的

  • 可以看到，在两个数据集上，匹配最好的词对始终是相同的

  • 差异在最差的词对上

    • The MEN words potatoes and tomato probably have low quality ImageNet-derived representations, because they occur often in the bottom pairs for that dataset.

    • The MEN words dessert, bread and fruit occur in the bottom 5 for both image datasets, which implies that their linguistic representations are probably not very good.

    • For WordSim353, the bottom pairs on ImageNet could be said to be similarity mistakes; while the ESP Game dataset contains more relatedness mistakes (king and queen would evaluate similarity, while stock and market would evaluate relatedness).

从以上的差异中难以得到某些结论，这是未来的一个研究方向
