（文献阅读）通用的 paraphrastic sentence embedding

发表于 2018-05-16

句子建模相关文献阅读

文献名称：Towards Universal Paraphrastic Sentence Embeddings

摘要（ABSTRACT）

本文研究了从释义数据库中有监督地学习通用释义句嵌入（paraphrastic sentence embeddings）的问题。本文比较了六种结构，根据标注文本相似性数据集对它们进行评价，这些数据集来自与训练数据相同的分布和广泛的其他领域。结果表明复杂的结构，比如 LSTM 在 in-domain 数据上表现最好；但是，在 out-of-domain 场景下，简单的结构，比如 word averaging 反而优于 LSTMs。本文设计的 averaging model 甚至与针对特定任务调优过的模型性能相当，但更简单易用。

为了更好地理解这些结构是如何比较的，本文在三个有监督的 NLP 任务上进行了实验：句子相似度（sentence similarity）、蕴涵（entailment）和情感分类（sentiment classification）。结果表明，词平均模型在句子相似性和 entailment 任务上表现优于 LSTMs。但是在情感分类任务上，LSTMs 表现非常好，甚至在斯坦福 Treebank 数据集上达到了 state-of-art。

此外，本文演示了如何将预训练的 sentence embedding 与这些有监督任务结合起来，并将它们用作先验（prior）和黑盒（black box）特征提取器，这使得在 SICK similarity 和 entailment 任务上达到 state-of-art。本文公开了所有资源在 http://ttic.uchicago.edu/˜wieting （无法打开，但是可以在作者的 GitHub 上找到源码）

前言（INTRODUCTION）

Word embeddings 介绍…

最近的工作开始寻求单词序列的表示（如短语、句子），并且出现了一些基础的结构，如对词向量求和（addition），RNN，CNN。

但是，很少有学习到的 sentence representations 能够像 word embedding 一样跨领域使用。

本文探讨了可以将任意单词序列编码成向量的组合模型，其生成的句向量表示在相似含义的句子上有较高的余弦相似性；重要的是，它可以方便地迁移到其他领域。

本文考虑了基于神经网络的六种组成结构，并基于来自 noisy phrase pairs from the Paraphrase Database (PPDB; Ganitkevitch et al., 2013) 来对它们进行训练。

这些结构包括从简单的求和模型到复杂的 LSTMs。

本文的一个重要发现是，将 vector averaging 加入模型并重新训练词向量，要比直接使用那些预训练好的词向量效果更好。

之前也看到不少，说直接将词向量求和/平均来获取 sentence embedding；但一直默认的是直接使用训练好的词向量，没意识到将求和/平均加入模型重新训练词向量。

除了在一系列实验上取得了不错的结果外，本文还演示了如何将利用 PPDB 语料训练的 sentence embeddings 与有监督任务结合起来，并利用他们作为一个 black box feature extractor。

相关工作（RELATED WORK）

最近很多工作开始关注如何学习 word sequences 的嵌入表示。他们使用的主要方法是对给定的词向量进行组合（compositionality）。因此，sentence embedding 的进展跟词向量是分不开的，从早期的统计特征（bi-gram）到现在大势所趋的神经网络。一些相关工作：neural bag-of-words models [1] (Kalchbrenner et al., 2014), deep averaging networks (DANs) [2] (Iyyer et al., 2015), …

这里我只列出了我比较感兴趣的

本文主要比较了 6 种结构（architectures）：

  1. word averaging
  2. word averaging followed by a single linear projection
  3. DANs [2]
  4. RNN
  5. LSTM without output gate
  6. LSTM with output gate

上述大部分工作都是在有监督的背景下学习组合模型（compositional models）。也就是说，”a training set is provided with annotations, and the composition function is learned for the purposes of optimizing an objective function based on those annotations.”

注意区分 composition function 和 objective function

而本文主要关注的是构建领域无关的 word sequences embeddings。一些相关工作：skip-thought vectors [3] (Kiros et al., 2015), … 这些工作提供了一种通用的表示方法，能够为文本分类或句子相似性任务提供某种特征。比如 skip-thought vectors 试图根据上下文来捕获句子间的相似性，而本文希望获得句子间的 paraphrastic similarity（释义相似性），即判断两句话是否有相同的含义。

释义识别（Paraphrase Identification）任务: https://aclweb.org/aclwiki/Paraphrase_Identification_(State_of_the_art)

本文的学习公式借鉴了大量有关学习输入表示的工作，以最大限度地提高学习空间中的相似性。(Our learning formulation draws from a large body of related work on learning input representations in order to maximize similarity in the learned space)

Weston et al., 2010; Yih et al., 2011; Huang et al., 2013; Hermann & Blunsom, 2014; Socher et al., 2014; Faruqui & Dyer, 2014; Bordes et al., 2014b;a; Lu et al., 2015; Wieting et al., 2015

模型（MODELS AND TRAINING）

本文的目标是将句子嵌入（embed）到一个低维空间，使空间中的余弦相似性对应于序列之间的解释关系的强度。

记单词序列 $x=[x_1,x_2,…x_n]$

模型 1

词向量求平均，训练参数为词向量矩阵 $W_w$，本文记该模型为 $g_{paragram-phrase}$

$$ g_{paragram-phrase}(x) = \frac{1}{n} \sum_i^n W_w^{x_i}
$$

$$ squash(x) = \frac{\left| x \right| ^2}{\left| x \right| ^2 + epsilon} \frac{x}{\left| x \right|}
$$

模型 2

在模型 1 的基础上添加一个投影矩阵（projection matrix），记为 $g_{proj}$

$$ g_{proj} = W_p (\frac{1}{n} \sum_i^n W_w^{x_i}) + b
$$

模型 3

来自 Deep Averaging Network(DAN) [2]，该模型可以看做是模型 2 使用多个隐藏层及非线性函数的推广。

模型 4

标准 RNN

$$ h_t = f(W_x W_w^{x_t} + W_h h_{t-1} + b)
$$

$$ g_{rnn}(x) = h_{-1}
$$

其中 $W_x$ 和 $W_h$ 为参数矩阵，$h_{-1}$ 表示词序列中最后一个单词的隐向量（hidden vector）

模型 5

一个特殊的 RNN——identity-RNN：In the identity-RNN, the weight matrices are initialized to identity, the bias is initialized to zero, and the activation is the identity function. We divide the final output vector of the identity-RNN by the number of tokens in the sequence. Thus, before any updates to the parameters, the identity-RNN simply averages the word embeddings. We also regularize the identity-RNN parameters to their initial values.

该模型的思想：The idea is that, with high regularization, the identity-RNN is simply averaging word embeddings. However, it is a richer architecture and can take into account word order and hopefully improve upon the averaging baseline.

模型 6

LSTM

本文注意到是否使用 output gate 会影响结果，因此该模型有两个版本，分别为是否使用 $o_t$。
